Embodied Interaction
A Kinesthetic Interface for Real-Time Audio Manipulation via Optical Hand Tracking

Stefanos Biliousis Aalborg University, Copenhagen, Denmark Medialogy MSc Embodied Interaction Semester Project
Section I

Project description

Interface that will manipulate audio effects in real time based on the position and movement of the user's hands. The system will use opencv/mediapipe and a webpacm (maybe leapmotion in the future) to track the user's hand and apply audio filters accordingly. The hand movements will be based on the LABAN framework. This is a project based on the Sonic Interaction Design project.

Relevant sources

    Towards an Interactive Tool for Music and Dance: Gestures, Laban Movement Analysis and Spectromorphology
